\item Open Game\+Project
\item Make sure Debug is set to x86
\item Under Solution Explorer, right click Game\+Project, then select Properties.
\item Under Configuration Properties, select Include Directories. Add the include files for all SDL2 Files.
\item Then Select Library Directories, add all lib x86 SDL2 Files.
\item Under C/\+C++, select Additional Include Directories, add Quick\+SDL.
\item Under Linker, select Additional Library Directories, add all lib x86 files under SDL2 Files.
\item Search windows \char`\"{}\+Edit the System Window Variables,\char`\"{} select the Advanced tab, then click Environment Variables.
\item Under System variables, select Path and click Edit. Add all lib x86 SDL2 Files.
